## Environment Variables

To run this project, you will need to add the following environment variables to your .env file in /server directory

`DB_TYPE`
`DB_PORT`
`DB_HOST`
`DB_USER`
`DB_PASS`
`DB_NAME`
`JWT_SECRET`
`ORIGIN` <--- set your client localhost server, e.g. http://localhost:8080

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
